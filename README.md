# sadiqk2.github.io

Personal portfolio / résumé site for Sadiq Khan.
Single-file, dependency-free, dark-mode aware — no build step.

**Live:** https://sadiqk2.github.io

## Edit

Everything is in `index.html`. To preview locally:

```bash
python3 -m http.server 8000   # then open http://localhost:8000
```

Push to `main` and GitHub Pages redeploys automatically within a minute or two.
